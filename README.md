<h1>EVO shop</h1>

<h3>EVO включает в себя</h3>
<ul>
    <li>Интернет магазин</li>
    <li>Блог</li>
    <li>Форум</li>
</ul>

<h3>Особенности интернет магазина</h3>
<h5>Программа лояльности:</h5>
<ul>
    <li>Кешбек счёт в личном кабинете</li>
    <li>Реферальная программа</li>
</ul>

<h3>Бэкенд:</h3>
<ul>
    <li>Язык Python</li>
    <li>Фреймворк Django</li>
    <li>БД Sqlite3</li>
</ul>
